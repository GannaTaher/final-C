## Description
A Clinic Management System that easy handle with Patients by 2 modes ,Written by C Language

## Features
Admin Mode 
    -Enter Patient
    -Edit Patient
    -Reserve a Slot
    _Cancel Reservation

User Mode
   -View Patient Record
   -View Today's Reservations
